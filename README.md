# VPN Telegram Bot

Telegram-бот для продажи VPN подписок с автоматической выдачей ключей через 3X-UI панель.

## Возможности

### Пользователь
- Личный кабинет (баланс, подписки)
- Покупка подписки (1/3/6/12 мес. со скидками)
- Выбор количества устройств (1-10)
- Тестовый период (настраивается в админке)
- Пополнение баланса через СБП
- Промокоды
- Партнерская (реферальная) программа
- Техподдержка

### Администратор (`/admin`)
- Управление серверами (добавление/удаление 3X-UI серверов)
- Настройка тарифов (цена, скидка)
- Управление пользователями
- Подтверждение/отклонение платежей
- Рассылка сообщений всем пользователям
- Настройка реквизитов оплаты (СБП)
- Настройка тестового периода
- Управление промокодами
- Статистика

### Автоматика
- Автоматическая выдача VPN ключей при оплате через 3X-UI API
- Поддержка протоколов VLESS и VMess
- Реферальные начисления

## Установка

### 1. Клонирование
```bash
git clone https://github.com/curcan-riwpyk-nikbE7/bot_vpn-.git
cd bot_vpn-
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка
```bash
cp .env.example .env
```
Отредактируйте `.env`:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=ваш_telegram_id
```

### 4. Запуск
```bash
python run.py
```

## Настройка серверов

1. Откройте бот, введите `/admin`
2. Нажмите "Сервера" → "Добавить сервер"
3. Введите название, URL панели 3X-UI, логин и пароль
4. Сервер будет автоматически подключен
